Markup
  • Markup tags are text elements that tell the browser how to treat the text.
  • They start with an opening tag.
  • They end with a closing tag.
  • The whole thing is called an element.
  • Example...b tag = Bold
  • Example...em tag = Emphasis
  • Example...p tag = Paragraph
  • Example...br tag = Line Break (VOID tag that creates white space)
  • Tags have in-line effects - text just breaks to the next line.
  • Blocks create an invisible box for text that has height and width.
  • SPAN is inline...DIV is block.
  • Computers are stupid so be precise with your code.
