2.6 Structured Data - Lists and For Loops

List Data Type (Structured Data)

  • Could contain anything we want (string, lists, numbers, etc).
  • Defined by [] with list elements seperated by calmas.
  • Can use positions [0:4] to return list values.

Nested Lists

  • You can mix up strings and numbers in lists.
  • You can also put lists within lists as list elements.
  • You can put anything you want in them.
  • If you break lists you should indent appropriately

Mutation

  1. We can change the value of a list after we create it.
  2. Variable reassignment changes the pointer to a new value.
  3. This does not change the value of the original value.
  4. In lists...we can change the values in a list.
  5. We refer directly to the list index and change the value at that index number.

Aliasing

  • Two different ways to refer to the same object.
  • Two variable names referring to the same object will result in changes to hte value of the object when either variable name introduces change.

List Operations

  • Append - This is a list method on the list. It will add a new element to the end of the list. stooges.append.
  • Plus - Combines lists into a single new list. list + LIst.
  • len - Short for length. You can pass in the operator and get the length of the list index of elements. len(list).
  • There is a big difference between Append and Plus.
  • Append add list elements (including other lists) to the list array.
  • PLUS combines lists into a single list array.

Loops on Lists

  • This includes WHILE loops and FOR loops.
  • FOR loop structure...for NAME in LIST: Do Something.
