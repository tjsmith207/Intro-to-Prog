1.1 The Basics of the Web and HTML

How the Web Works

  • The web is made up of you, your computer/browser, the Internet, HTTP, and servers.
  • The web is a collection of HTML (Hypertext Markup Language) documents.
  • Hyperlinks in the HTML documents glue the web together.
  • The web is made up of approximately 30 billion pages.

HTML Markup

  • Markup tags are text elements that tell the browser how to treat the text.
  • They have opening and closing tags with the whole called an element.
  • Tags have in-line effects, while blocks create an invisible box with height and width.
  • SPAN is an inline example, while DIV is a block example.
  • Computers are stupid so be precise with your code.

HTML Document Structure

  • DOCTYPE - Indicates the document type.
  • thtml - Indicates the markup type.
  • head - Location for Meta Data, Java Script, CSS.
  • title - Determins the title of the document.
  • Body - The visible part of the HTML.
