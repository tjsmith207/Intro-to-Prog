1.3 Adding CSS Style and HTML Structures

Cascading Sytle Sheets

  • Singe source for styling.
  • Avoid repetition to create good code.
  • Could be in-line or external file.
  • Rule "Cascade" so are applied top down from code.
  • Elements can be selected by tag or class.
  • Link to Mozilla CSS Reference

CSS Language

  • A Selector (Period followed by selector)
  • Curly Braces
  • Declarations with Properties and Values
  • Properties (ie. background-color)
  • Value (ie. black)

The Box Model

  • Everything on the web is a box.
  • Box includes Margin, Border, Padding, and Content.
  • Box-Sizing: Border-Box; controls max box size.
  • Using percent (pc) resizes the box based on screen size.
  • Using pixel (px) fixes the box size.
  • Divs are block elements.
  • Adding rule "display: flex;" - boxes next to each other.
