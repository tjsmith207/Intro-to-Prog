1.2 Creating a Structured Document

Purpose of HTML, CSS, and JavaScript

  • Front end development is visual.
  • It is how users interact with your application.
  • HTML files are the structure of the document.
  • CSS are the style (colors, decorations, etc.).
  • JavaScript creates interactive components.

HTML (HyperText Markup Language)

  • The standard markup language for creating web pages.
  • Has syntax and rules.
  • Controls the "structure" of a page.
  • Tags are the basic element of HTML.
  • Tags can have classes and values.

CSS (Cascading Style Sheets)

  • Language used to describe document look and formatting.
  • Controls the "style" of a page (how it looks).
  • Allows you to change the look of elements on the page.
  • w3schools CSS Introduction

DOM (Document Object Model)

  • The browser built, tree-like structure of the page.
  • The tree contains tags, elements, anchors, etc.
  • Cross-platform and language independent.
  • w3schools DOM Reference

Page Breakdown

  • Chrome Developer Tools
  • Everything on a web page is a box.
  • Break a webpage into the boxes it takes to create the page.
  • The breakdown is called "Boxifying Design".
  • Work from big to small.
  • Practice "Technical Empathy to see the page from the computers perspective.
